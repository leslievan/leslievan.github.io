使用中科大镜像源加速 Conda#

由于 Windows 文件系统限制，无法在资源管理器中直接创建名为 .condarc 的文件，可先执行 conda config --set show_channel_urls yes，该命令将在用户目录 $HOME 下生成 .condarc 文件。

打开 $HOME/.condarc 文件，将内容修改为：

ssl_verify: false
show_channel_urls: true
channels:
  - https://mirrors.ustc.edu.cn/anaconda/pkgs/main
  - https://mirrors.ustc.edu.cn/anaconda/pkgs/r
  - https://mirrors.ustc.edu.cn/anaconda/pkgs/msys2
  - defaults

保存后重启终端配置即可生效，执行 conda clean -i 清理缓存。

使用中科大镜像加速 pip#

执行 pip config set global.index-url https://mirrors.ustc.edu.cn/pypi/web/simple，该命令会生成 $HOME/AppData/Roaming/pip/pip.ini 文件，重启终端配置即可生效。

** pip install 安装包时提示 SSL 错误**

添加信任的主机，编辑 $HOME/AppData/Roaming/pip/pip.ini，替换为：

[global]
index-url = https://mirrors.ustc.edu.cn/pypi/web/simple

trusted-host = pypi.python.org
               mirrors.ustc.edu.cn
